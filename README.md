# Hospital Management System (Java Swing)

A simple hospital management desktop application built using Java Swing. This system allows adding patient records, assigning doctors, calculating bills, and managing hospital data in one place.

## Features
- Add and manage patient records
- Add doctors and assign patients
- Room allocation
- Admitted / not admitted option
- Bill calculation based on days
- GUI using Java Swing
- Simple table based record listing

## Technologies Used
- Java
- Swing
- AWT
- JTable, DefaultTableModel

## How to Run
1. Open the project in VS Code / IntelliJ
2. Compile and run `HospitalManagementSystem.java`
3. Login using basic credentials (admin section)
